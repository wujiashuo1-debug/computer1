## 上传 GitHub Pages

把压缩包解压后，将文件夹内部所有内容上传到仓库根目录：

```text
index.html
style.css
app.js
questions.js
manifest.webmanifest
sw.js
assets/
README.md
打开网站说明.txt
```

不要把外层文件夹整体套进去。
